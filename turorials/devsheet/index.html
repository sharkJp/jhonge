DevSheet Pro

Extensión VS Code

¿Qué es?

DevSheet es una extensión de VS Code para desarrolladores que permite guardar, organizar y reutilizar comandos personalizados por lenguaje de programación, directamente dentro del editor, sin depender del navegador y aumentar la productividad.Tambien trabaja de forma local, sin necesidad de conectarse a internet.

Tecnologías

  • HTML, CSS, JavaScript
  • VS Code API
  • Web Components
